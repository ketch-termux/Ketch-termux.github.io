Pushforce

Verificador para entornos Termux

Este script verifica si tu entorno en Termux está listo para usar pushforce.sh. Detecta Git, tu configuración, credenciales de GitHub y si el alias push está disponible.

Modo demostración

-test   # Simula todo el proceso sin ejecutar comandos

Ejecutar sin descarga

Con curl

bash <(curl -s https://raw.githubusercontent.com/ketch502m/pushforce/main/verify.sh)

Con wget

bash <(wget -qO- https://raw.githubusercontent.com/ketch502m/pushforce/main/verify.sh)

Modo prueba (-test)

bash <(curl -s https://raw.githubusercontent.com/ketch502m/pushforce/main/verify.sh) -test
bash <(wget -qO- https://raw.githubusercontent.com/ketch502m/pushforce/main/verify.sh) -test

Alias recomendado

Agrega lo siguiente a tu archivo ~/.bashrc o ~/.zshrc:

alias verifypush='bash <(curl -s https://raw.githubusercontent.com/ketch502m/pushforce/main/verify.sh)'

Y luego ejecuta:

verifypush -test

Requisitos del sistema

  • Termux con bash
  • Conexión activa a Internet
  • curl o wget

¿Qué valida el script?

  1. Que Git esté instalado (y lo instala si no lo está).
  2. Que tengas nombre y correo en Git configurados.
  3. Que poseas un token personal de GitHub.
  4. Que tengas el alias push apuntando a pushforce.sh.

¿Cómo obtener pushforce.sh?

1. curl

curl -o ~/pushforce.sh https://raw.githubusercontent.com/ketch502m/pushforce/main/pushforce.sh

2. wget

wget -O ~/pushforce.sh https://raw.githubusercontent.com/ketch502m/pushforce/main/pushforce.sh
chmod +x ~/pushforce.sh

3. Clonando el repositorio

git clone https://github.com/ketch502m/pushforce.git
cd pushforce && chmod +x pushforce.sh verify.sh
mv pushforce.sh ~/pushforce.sh
echo "alias push='bash ~/pushforce.sh'" >> ~/.bashrc && source ~/.bashrc
